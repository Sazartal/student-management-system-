WASTE MANAGEMENT SYSTEM


Student: Salma Nazaralieva

Description  
This project is a simple console application written in Java.  
It allows the user to manage waste records.  
The user can add, view, delete, and update records.  
Each record includes date, type, amount, and location.  

The program uses a menu system and stores data in a file so that records are saved between runs.


Objectives  
The goal of this project is to practice object-oriented programming and build a working system that can manage data using CRUD operations.  
Another goal is to apply validation, file handling, and program structure.

Project Requirements Implemented  
1. CRUD operations (create, read, update, delete)  
2. Command line interface with menu  
3. Input validation for date and numbers  
4. Data saved in file (data.txt)  
5. Modular structure with multiple classes  
6. Documentation in README file  
7. Error handling using try-catch  
8. Encapsulation (private fields, getters)  
9. Inheritance (base and child classes used)  
10. Polymorphism (methods used in different forms)  

Technologies Used  
Java  
IntelliJ IDEA  
Scanner  
ArrayList  
File I/O  

Project Structure  
Main.java – program start and menu  
WasteManager.java – main logic (add, delete, show, update)  
WasteRecord.java – data model of record  
Record.java – input and validation  
data.txt – file for storing data
